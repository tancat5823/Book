# Book
This is a book tracking application. It can read files containing formatted information of a book and create printed book and audio book objects. From there the user can sort, search, and add books.
To run the program, create a test file or use the supplied BookTest file. You can use this project with files formatted like the sample TXT Project file.txt i.e. in the order of Title,Author,Genre,BookType,Pages,Duration_Minutes,Cost. The book interface is an interface that defines some of the key functionallities needed. The Book class is an abstract class that further defines some of the key functionas required. The printed book and audio book classes define how to create book objects of their respective type and are used to store the data of books from the file. Finally the book application handles some of the user facing functions like sorting, searching, adding, etc. The code can be ran easily by importing the BookInterface, Book, PrintedBook, AudioBook, and BookTest files in a folder and running the BookTest file in your IDE.
The project is designed so that the PrintedBook and AudioBook classes can be used to create instances of book objects. These books can then be stored and handled by the book application. The test file just calls these functions in response to human input.
